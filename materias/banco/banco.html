BANCO DE DADOS

1° Trimestre


Não possui atividades.

2° Trimestre


Envolveu a prática de construção de modelos lógicos em banco de dados, focando na organização e definição de tabelas, colunas e relacionamentos. Esse exercício ajuda a estruturar dados de maneira lógica para facilitar seu uso e gerenciamento no banco de dados.

H1 e H2

A atividade envolveu uma pesquisa sobre o conceito de normalização de dados em banco de dados, explorando técnicas para organizar dados de forma a reduzir redundâncias e melhorar a integridade.

H3

A atividade foi uma apresentação de slides para desenvolver modelos conceitual e lógico de um banco de dados para um consultório médico, definindo entidades e relacionamentos essenciais para organizar as informações do sistema.

H1 e H2

3° Trimestre


A atividade envolveu o desenvolvimento do banco de dados de uma biblioteca no modelo físico, detalhando tabelas, tipos de dados, restrições e índices para estruturar o armazenamento e o acesso eficiente às informações.

H4

A atividade abordou o uso de LEFT JOIN e RIGHT JOIN em SQL, focando no modelo físico de banco de dados, para combinar dados de tabelas diferentes, incluindo todos os registros de uma tabela e correspondências específicas da outra.
